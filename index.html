O que são Clients

Conceito de client

Dispositivo que exige determinados serviços de um servidor.

Falando sobre navegadores

São programas criados por empresas, utilizados para abrir/executar arquivos.

Seguem padrões W3C.

Também chamados de "browsers".

Iniciou com o MOSAIC, passou para o Netscape, e hoje, temos uma variedade de navegadores disponíveis e gratuitos.

Aplicações Web

Resumindo: são soluções criadas que possuem a internet como meio de comunicação entre Client x Server, não sendo necessária sua instalação.

Linguagens de Programação

Linguagem de programação: o que é?

Linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas(softwares).

Linguagem Server-Side

Linguagens que são executadas no servidor.

Exemplos: JavaScript, Java, PHP, C# e Ruby.

Linguagem Client-Side

Linguagens que são executadas no client.

Exemplo: JavaScript.

Tags

O que são tags HTML?

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Tipos de tag

As tags podem ser categorizadas inicialmente em dois tipos, em “nível de bloco” (block-level) e “em linha” (inline). Um elemento em nível de bloco ocupa toda a largura de seu elemento pai, que também chamamos de elemento container, criando assim um “bloco”. Já os elementos inline, geralmente usamos para demarcação de conteúdos de texto.

Exemplos de Tags

  • < !DOCTYPE html >
  • < html >
  • < body >
  • < script >
  • < head >
  • < meta >
  • < link >
  • < style >

Tags semânticas

Tags semânticas são tags que possuem um significado, que dão sentido a informação de texto ao navegador e buscadores. É uma boa prática tentar sempre utilizar essas tags semânticas para ajudar no entendimento do código, além de ajudar muito no SEO do site

Listas Ordenadas e Não Ordenadas

A lista mais comum é a lista não-ordenada definida pela tag < ul > (unordered list). Para cada item da lista não-ordenada, utilizamos uma marcação de item de lista < li > (list item). A mesma tag de item de lista < li > é utilizada quando demarcamos uma lista ordenada.

As listas ordenadas (< ol > - ordered list) também podem ter sua estrutura composta por outras listas ordenadas. Também é possível ter listas ordenadas aninhadas em um item de uma lista não-ordenada e vice-versa.

Links

Quando precisamos indicar que um trecho de texto se refere a um outro conteúdo, seja ele no mesmo documento ou em outro endereço (por exemplo uma página na web), utilizamos a tag de âncora < a >.

Existem três diferentes usos para as âncoras: definição de links, bookmark e demarcação de destinos para links dentro do próprio site.
